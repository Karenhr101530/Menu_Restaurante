# Gestión de Precios de Menú de Restaurante

## Descripción
Este proyecto fue desarrollado como solución al problema propuesto en la Fase 5 - Evaluación Final POA.

El programa permite gestionar un menú de restaurante y calcular el precio final de los productos aplicando una promoción específica.

## Funcionalidad
- Almacena los productos del menú en una lista.
- Recorre todos los productos mediante una estructura repetitiva.
- Utiliza una función para calcular el precio final.
- Aplica un descuento del 15% a los productos de la categoría "Plato Fuerte" cuyo precio base sea superior a $30.000.
- Muestra en consola el nombre del producto, categoría, precio base y precio final.

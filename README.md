# Gestión de Precios de Menú de Restaurante

## Descripción
Este proyecto fue desarrollado como solución al Problema 2 de la Fase 5 - Evaluación Final POA.
El programa gestiona los precios de un menú de restaurante y aplica una promoción automática
a productos específicos usando el paradigma de programación estructurada.

## Funcionalidad
- Almacena los productos del menú en una matriz con la estructura [Nombre, Categoría, Precio Base].
- Recorre la matriz completa mediante una estructura repetitiva (ciclo for).
- Utiliza una función como módulo para calcular el precio final de cada producto.
- Aplica un descuento del 15% a los productos de la categoría "Plato Fuerte" cuyo precio base supere $30.000.
- Mantiene el precio base sin cambios si no se cumplen las condiciones de la promoción.
- Muestra en consola el nombre, categoría, precio base y precio final de cada producto.
